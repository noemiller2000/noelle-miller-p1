Let's make a Mad Lib!

This website uses your input to make a short mad lib, or a story created from words the user provides that usually results in hilarious nonsense.
See the instructions below to get started.

  1. In the table below, click the button to the first word
  2. Enter in a word of the specified type (i.e. noun, adjective, verb, etc.)
  3. Rinse and repeat until a value has been entered for each row
    • Feel free to reenter a value for a word if you don't like your original answer
  4. Press the generate button to see your story come to life!

Please don't refresh the page before you are done with the Mad Lib, as it will erase your progress!
